Product images slider for thumbnails image navigator in detail product page.

== Description ==

A WooCommerce extension that display thumbnails slider when the product has many gallery images or many variable product images.
It is smooth and light, incredible, responsive display. It also does not needs to change woocommerce js core and template files to work.
WooCommerce Thumbnails Slider is working with all free theme and all premium themes.

Beside, the plugin also provide a admin settings to helps user customs the thumbnails slider easily.

== Installation ==

1. Simply install as a normal WordPress plugin and activate.    
2. Settings page, go to WooCommerce/Settings/Thumbnails Slider


== Frequently Asked Questions ==

= Does this plugin work fine with variable products that has many image?

 Sure, it works fine.

= How i can config some option in slider? =
 
 A admin settings page is providing in WooCommerce/Settings/Thumbnails Slider
